Aprende a construir páginas web modernas con HTML, CSS y JavaScript desde cero.

🧱 ¿Qué es HTML?

HTML es la estructura de una página web. Es como el esqueleto.

Vista en navegador:

Título de la página

Este es un párrafo

Código:
<h1>Título de la página</h1>
<p>Este es un párrafo</p>

🎨 ¿Qué es CSS?

CSS se usa para darle estilo a la página: colores, tamaños y diseño.

Hola Mundo

Código CSS:
h1 {
color: #00c8ff;
font-size: 30px;
}

⚡ ¿Qué es JavaScript?

JavaScript agrega interacción y lógica a la página web.

Código:
alert("Hola desde JavaScript");
