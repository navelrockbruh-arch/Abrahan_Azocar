🚀 ¿Qué es Flutter?

Flutter es un framework de Google para crear apps móviles multiplataforma con un solo código.

Hola App 📱
import 'package:flutter/material.dart';

void main() {
runApp(MaterialApp(
home: Text("Hola App")
));
}
